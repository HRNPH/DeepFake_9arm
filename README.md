# DeepFake_9arm
 DeepFake_Dataset for 9arm using DeepFaceLab (trained using fast96)

## Included Datasets
- 9arm face(masked)
- Des face(masked)
## Usage
- place it in the same dat_src or dat_dst /align folder to skip face extraction process
